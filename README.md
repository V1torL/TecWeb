# Configurando
Para inicializar o banco:

```
  npm run db:init
```

É preciso setar a variável de ambiente `AUTH_SECRET` (https://authjs.dev/getting-started/installation?framework=Next.js).

Você pode criar um arquivo `.env` e executar

```
  npx auth secret
```

# Executando
Basta rodar:
```
  npm run dev
```
